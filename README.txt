Console Calculator

A simple calculator application built in Java that performs basic arithmetic operations right in the console.
This is my first GitHub project, and it reflects my journey into programming and version control with Git!

Features : 
1) Addition
2) Subtraction
3) Multiplication
4) Division (with division by zero handled gracefully)
5) Calculate Square Roots

How to run? 
1) Clone this repository to you local machine.
2) Navigate to the project Directory.
3) Compile the program
4) Run the Program.

What I Learned : 

1) Using java for simple console based applications.
2) Applying Git for version control and managing projects.
3) Pushing a project to GitHub for the first time.
